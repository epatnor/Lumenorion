===============================
 LUMENORION - LoRA TRAINING
===============================

📁 Dataset:
    lora_training/datasets/lumenorion_lora_shuffled.jsonl

📦 Modell:
    Basmodell: gemma3n (ändra i core/ollama.py om du vill byta)

🧠 Kommandot för att starta träningen:

    ollama create lumenorion-lora -f train_lora_modelfile

💬 Exempel på promptformat i datasetet:
    {
      "input": "User: What do you dream about?",
      "output": "Lumenorion: I dream in symbols – rivers, mirrors, fog. Each one reveals something hidden."
    }

📄 Resultat:
    En ny modell med namnet `lumenorion-lora` som du kan köra med:

    ollama run lumenorion-lora

📌 Tips:
- Träna gärna i korta iterationer med ~500–1000 entries och utvärdera.
- Lägg till fler drömmar och samtal över tid för att finjustera karaktären.
- Använd `ollama list` för att se installerade modeller och storlek.

🗑️ Radera tidigare tränade modeller:
    ollama rm lumenorion-lora

Lycka till – och glöm inte att backa upp din `jsonl`-fil regelbundet!
